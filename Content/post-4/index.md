#### Zip a file
```
	gzip filename
```
#### Unzip a file
```
	gunzip filename.gz
```
#### Compress a folder
```
	tar cvf name_of_compressed_file.tar  folder_name_or_list_of_files
```
#### Extract a file
```
	tar xvf filename.tar
```
